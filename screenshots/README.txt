How to create a screenshot for the documentation:

1. Run the application by executing:
   python main.py

2. Take a screenshot of the application window showing:
   - The main interface with input fields
   - Settings with model size and language selections
   - A video file selected (if possible)

3. Save the screenshot as "screenshot.png" in the docs folder

4. Update the GitHub repository with the screenshot

The screenshot will be used in the README.md documentation to give users a visual preview of the application.
